  • H5

    H5是指第五代HTML,它构成了互联网兴起的基础，它最显著的优势在于跨平台性，强大的兼容性显著的降低了开发与运营成本，让企业特别是创业者获得更多的发展机遇。

  • CSS3

    css3语言是朝着模块化发展的，更多新的模块也被加了进来，比如：盒子模型，超链接方式，文字特效，背景和边框

  • Javascript

    javaScript是一种解释性脚本语言，主要用来向HTML页面添加交互行为.

  • jQuery

    它是继Prototype之后又一个优秀的javaScript代码库，它具有独特的链式语法和短小清晰的多功能接口。

  • php

    它是一种通用开源脚本语言，主要适用于Web开发领域，它可以更快速的执行动态网页，还可以执行编译后代码，使代码运行更快。

  • AngularJS

    AngularJS 是一个 JavaScript 框架。它是一个以JavaScript编写的库。它可通过script标签添加到HTML页面。
